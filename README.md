Jal Jeevan Mission – Android Application

An Android application built using Java, XML, and Firebase to support digital monitoring of water supply progress under the Jal Jeevan Mission. The app helps track village details, water sources, household tap connections, and implementation progress in a simple and efficient way.

⭐ Project Overview

The Jal Jeevan Mission Android App is designed to help field workers, engineers, and administrators record and access real-time data. The goal is to replace manual record-keeping with a digital, centralized system.

The app allows users to enter village information, update connection status, monitor pipeline progress, store water source details, and track daily activities using Firebase.

🚀 Features
User Authentication (Firebase Authentication)
Secure login for staff and admin users.
Add & Manage Villages
Store village name, population, number of households, and coverage progress.
Household Tap Connection Tracker
Update which households have received functional tap connections.
Water Source Information
Add details about borewell, tank, handpump, reservoir, etc.
Pipeline Progress Monitoring
Record total length, completed work, and pending work.
Real-Time Data Sync (Firebase Database / Firestore)
All data updates instantly for all users.
Simple UI & Easy Navigation
Designed using XML layouts for clarity and usability.
🛠️ Tech Stack
Frontend: Java, XML
Backend: Firebase Authentication, Firebase Realtime Database / Firestore
Tools: Android Studio, Gradle

Note: No Firebase Cloud Storage is used.
